AWS Cost Optimization Dashboard

AWS Python React Docker Terraform PostgreSQL

Open-source tool for monitoring and optimizing AWS cloud spending

View on GitHub Live Demo

Overview

This project addresses the challenge of growing cloud costs by providing a comprehensive dashboard that monitors AWS spending, identifies optimization opportunities, and provides actionable insights.

Key Features

  • Real-time cost monitoring across multiple AWS accounts
  • Automated identification of idle resources
  • Right-sizing recommendations for EC2 instances
  • Reserved Instance planning and suggestions
  • Customizable alerts for budget overruns
  • Monthly savings reports with PDF export

Technical Architecture

  • Backend: Python Flask API with AWS Cost Explorer integration
  • Frontend: React dashboard with Chart.js for visualizations
  • Database: PostgreSQL for storing historical cost data
  • Infrastructure: Terraform for AWS resource provisioning
  • Deployment: Docker containers on AWS ECS
  • CI/CD: GitHub Actions for automated testing and deployment

Results & Impact

  • Identified 35% potential cost savings for pilot organizations
  • Reduced manual cost analysis time from 8 hours to 15 minutes weekly
  • Implemented automated reporting saving 40+ hours monthly
  • Used by 50+ AWS customers to optimize their cloud spending

Technologies Used

AWS Cost Explorer API, Terraform, GitHub Actions
